### Modul 1: JavaScript Dasar (2-3 sesi)
**Topik:**

### Modul 2: TypeScript (2-3 sesi)

- Type Inference

**Fokus:** Type safety untuk development yang lebih baik

**Fokus:** Version control untuk development modern

## ğŸ“– Cara Menggunakan

### Untuk Instruktur

1. **Sebelum Mengajar:**
   - Baca `PANDUAN-INSTRUKTUR.md`
   - Review setiap modul yang akan diajarkan
   - Siapkan contoh kode untuk demonstrasi
   - Setup environment (Node.js, PostgreSQL, dll)

2. **Saat Mengajar:**
   - Ikuti struktur modul
   - Gunakan "Penjelasan untuk Instruktur" sebagai panduan
   - Demonstrasi dengan kode langsung
   - Berikan waktu untuk latihan

3. **Setelah Mengajar:**
   - Review apakah peserta memahami
   - Berikan feedback
   - Siapkan untuk sesi berikutnya

### Untuk Peserta

1. **Ikuti Urutan Modul:**
   - Jangan skip modul
   - Pahami setiap konsep sebelum lanjut
   - Kerjakan semua latihan

2. **Praktik Langsung:**
   - Tulis kode sendiri, jangan hanya membaca
   - Test setiap contoh kode
   - Eksperimen dengan variasi kode

3. **Gunakan Repository Boilerplate:**
   - Sebagai referensi di modul akhir
   - Pelajari struktur project
   - Implementasikan fitur baru
